
# DNC-ecomerce-bd

Bamco de dados do ecomerce da DNC de produtos de beleza

## Instalação

Fazer a seguintes instalações

```bash
  npm i express
  npm i nodemon
  npm i mongoose
```
    
## Funcionalidades

- Adiciona um novo cliente
- Adiciona um novo produto
